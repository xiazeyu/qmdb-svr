# qmdb-svr

## Quick Installation
`npm install`
import `db_dump.sql` to localhost MySQL server

## Start
change `env=production` to `env=development` in `.env` if needed.
`npm install`
`npm start`

## Miscs
- build swagger doc: `npm run build-docs`
- lint code: `npm run lint`
- lint and fix code: `npm run lint-fix`
